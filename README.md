# Teslo API

1. Clonar proyecto
``` npm install ```
2. Clonar el archivo ```.env.template``` y renombrarlo a ```.env```
3. Cambiar las variables de entorno
4. Levantar base de datos
```docker-compose up -d```
5. Ejecutar SEED: ```http://localhost:3000/api/seed```
6. Levantar: ```npm run start:dev```
